### BEMHTML playground
https://rozhdestvenskiy.ru/dev/bemhtml/

#### Development
```bash
git clone https://github.com/sbmaxx/bemhtml-demo.git --depth=1
cd bemhtml-demo
npm install
npm start
```

#### Production
```bash
git clone https://github.com/sbmaxx/bemhtml-demo.git --depth=1
cd bemhtml-demo
npm install
npm run build
```
